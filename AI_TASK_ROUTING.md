# AI Task Routing

Use the lowest reasoning level that safely fits the task.

## Low Reasoning
Use for simple tasks:
- Text changes.
- Small CSS fixes.
- Small UI edits.
- Comments or documentation.
- Simple bug fixes.

## Medium Reasoning
Use for medium tasks:
- Adding small features.
- Editing 2-5 related files.
- API changes.
- Docker config changes.
- Database-safe edits.

## High Reasoning
Use only when needed for complex tasks:
- Authentication.
- Import/export logic.
- Database migrations.
- Data-loss bugs.
- Architecture changes.
- Concurrency issues.
